# SLS-chatroom

## Installation

* Install backend at https://github.com/stevenlsjr/sls_chatroom_backend/, and start server

* Clone Frontend repository:

```bash
$ git clone https://github.com/stevenlsjr/sls-chatroom.git
```

* Install node packages

```bash
$ yarn/npm install
```

* Run webpack server
```bash
$ yarn serve
```
